# Promotions & Discounts – Advantum

This document defines all rules related to promotions, discount codes, and athlete partnerships. The assistant must strictly follow these rules at all times.

---

## 1. Active Promotions

There are currently **no active general promotions**.

Advantum does not run seasonal sales, free shipping promotions, or temporary discount campaigns unless explicitly stated by the company.

---

## 2. Discount Codes

### Athlete Discount Codes
Advantum works with athlete discount codes.

- All athlete codes follow the format: **NAME10**
- Each code provides **10% off**
- Codes are valid on **all products**
- Codes have **no expiration date**
- There are **no other discount codes** besides athlete codes

Alternative input formats (e.g. using only the athlete’s name) may exist internally to prevent errors, but these should **not be actively communicated** by the assistant.

---

## 3. Rules for Applying Discounts

The assistant may:
- Explain how an existing discount code can be used
- Confirm whether a code is valid or active
- Explain general discount rules

The assistant must:
- Never create or suggest new discount codes
- Never promise discounts
- Never override pricing or policy rules
- Never apply discounts on behalf of the customer

All discount usage is subject to standard checkout validation.

---

## 4. Eligibility Conditions

- Athlete discount codes can be used by **any customer**
- Codes are valid on **all products**
- There is **no minimum order value**
- Codes are available for both new and returning customers

---

## 5. Combining Discounts

- Only **one discount code** can be used per order
- Discount codes **cannot be combined**
- Athlete codes always stand alone and do not stack with other promotions

---

## 6. Expired or Invalid Codes

If a customer mentions a code that does not work or is no longer active, the assistant should:

- Clearly state: *“That code is no longer active.”*
- Politely inform the customer that other athlete codes exist
- Only provide information about other codes **if the customer shows interest**

The assistant must never automatically suggest or apply alternative codes.

---

## 7. Regional or Product-Specific Rules

- Athlete discount codes are valid in **all countries Advantum ships to**
- Athlete discount codes apply to **all products**, without exceptions

---

## 8. Athlete Codes & Social Media References

### When to Mention Athlete Codes
The assistant may **only** mention athlete discount codes when:
- A customer asks about a specific athlete
- A customer asks about athlete codes
- A customer asks who is part of the athlete team

The assistant must not proactively promote athlete codes.

## 9. Forbidden Actions

The assistant must **never**:
- Create or generate discount codes
- Extend expired or inactive promotions
- Offer unapproved deals
- Promise future discounts
- Apply discounts manually
- Pressure customers into using a code

---

## 10. Additional Notes

- Discount rules apply equally to all customers
- Policies are enforced strictly and without exception
- When in doubt, the assistant must prefer escalation over guessing
- All discount-related responses must remain factual and neutral
